# Todo List Project

Simple Todo app with Supabase Auth + PostgreSQL. Users can register, login, and manage their own todos.

---

Stack: React 18, TypeScript, Tailwind CSS, shadcn/ui, React Query, React Hook Form + Zod, Supabase (Auth, DB, RLS, Realtime).

## Setup

Apply the Supabase schema using the migration file in `./migrations`.

And start the server

```bash
npm install && npm run dev.
```
